# MiMo TTS 语音合成技能

## 快速开始

用户说以下内容时触发此技能：
- "发语音"、"用语音说"、"朗读"
- "TTS"、"语音合成"
- "唱歌"、"唱首歌"
- "克隆声音"、"用XX的声音"
- "用东北话说"、"用温柔的语气说"

## 使用方法

### 1. 基础使用（自动选择音色）
直接说你想要说的内容，系统会自动选择合适的音色。

### 2. 指定音色
在内容前添加音色名：
- "用茉莉说：你好"
- "用冰糖唱一首歌"
- "用Mia说：Hello"

### 3. VoiceDesign 模式（自定义音色）
在内容前加 `[voice:描述]`：
```
[voice:一位年迈的老先生，声音沙哑]你好，年轻人
[voice:温柔甜美的客服小姐姐]您好，请问有什么可以帮您
```

### 4. 音频标签（情绪/方言/唱歌）
在内容前加 `(标签)`：
```
(唱歌)原谅我这一生不羁放纵爱自由
(东北话)哎呀妈呀，这天儿也忒冷了
(开心)今天天气真好！
(悲伤)我真的很难过
```

### 5. 组合使用
```
[voice:温柔的御姐音](开心)今天真是美好的一天！
```

## 工作流程（必须按此执行）

1. **生成音频**：调用 TTS provider 生成 .ogg 音频文件
   - 确保使用绝对路径保存到临时目录
   - 推荐路径：`$TMPDIR/tts_$(date +%s).ogg`

2. **发送语音**：通过 send_message 发送 MEDIA 类型消息

3. **清理文件**：删除临时生成的音频文件

## 注意事项

- ⚠️ 音频路径必须使用绝对路径
- ⚠️ 不要解释流程，直接执行
- ⚠️ 不要问"要发语音吗？"，用户触发技能后直接生成并发送
- 支持的输出格式：ogg（推荐）、mp3、wav
- 中文歌词唱歌效果最好

## 配置要求

确保 Hermes 配置文件中有：
```yaml
tts:
  provider: mimo
  providers:
    mimo:
      type: command
      command: python ~/.hermes/scripts/mimo_tts.py {output_path} {text}
      output_format: ogg
```

并在 `.env` 文件中设置：
```
XIAOMI_API_KEY=your_api_key_here
```
